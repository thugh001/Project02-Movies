Movie Explorer: Explore The Movie Dataset, Review Movie Analytics, Search For Movie Information, And More!


Project Group: Kamila Bajaria, Negin Nickparsa, Tony Fernandez, Tashia Hughes


The dataset used for this visualization comes from Kaggle and IMDB. The Kaggle metadata.csv file contains 45,000 movies released on or before July 2017. Data points include budget, revenue, release dates, languages, and more. The IMDB data was joined with the IMDB ID from Kaggle to pull in film location.


This dashboard will allow you to visualize and interact with the dataset, compare revenue for the 100 highest grossing movies, and interact with a leaflet map to review the location where a movie was filmed.
